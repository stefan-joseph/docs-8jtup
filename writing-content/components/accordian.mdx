The Accordian component allows you to create collapsible sections in your documentation. Users can expand and collapse content to focus on what they need, making it perfect for FAQs, detailed explanations, or organizing complex information.

## Basic Usage

<Demo>
  <UiDisplay>
    <Accordian title="Getting Started">
      To get started with our framework, first install the package:

      ```bash
      npm install my-framework
      ```

      Then import and initialize the framework in your project.
    </Accordian>

    <Accordian title="Configuration">
      Configure the framework by creating a config file:

      ```javascript
      // config.js
      module.exports = {
        theme: 'dark',
        debug: process.env.NODE_ENV === 'development'
      };
      ```
    </Accordian>

  </UiDisplay>

## With Icons

You can add icons to your accordions using the `icon` prop. Icons are sourced from Lucide and help make your accordions more visually distinctive.

<Demo>
  <UiDisplay>
    <Accordian title="Installation" icon="download">
      Install the package using your preferred package manager:

      ```bash
      npm install my-package
      # or
      yarn add my-package
      ```
    </Accordian>

    <Accordian title="Configuration" icon="settings">
      Configure your settings in the config file:

      ```json
      {
        "theme": "dark",
        "language": "en"
      ```

## Accordian Groups

Use `AccordianGroup` to create a set of accordions where only one can be open at a time. This is perfect for FAQs or mutually exclusive options.

<Demo>
  <UiDisplay>
    <AccordianGroup>
      <Accordian title="What is this framework?">
        Our framework is a modern, lightweight solution for building web applications with excellent developer experience and performance.
      </Accordian>

      <Accordian title="How do I get started?">
        Getting started is easy! Install the package, create a new project, and follow our quickstart guide in the documentation.
      </Accordian>

      <Accordian title="Is it free to use?">
        Yes, our framework is open source and free to use for both personal and commercial projects under the MIT license.
      </Accordian>
    </AccordianGroup>

  </UiDisplay>
