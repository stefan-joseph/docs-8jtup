The Tabs component allows you to create tabbed interfaces in your documentation. Users can click between tabs to view different content, making it perfect for organizing related information or showing different options.

## Basic Usage

<Demo>
  <UiDisplay>
    <Tabs>
      <Tab label="Installation">
        To get started, install the package using your preferred package manager:

        ```bash
        npm install my-package
        # or
        yarn add my-package
        # or
        pnpm add my-package
        ```

        After installation, you can import and use the component in your project.
      </Tab>
      <Tab label="Configuration">
        Configure the component by passing props:

        ```jsx
        import { MyComponent } from 'my-package';

        function App() {
          return (
            <MyComponent
              theme="dark"
              size="large"
            />
          );
        }
        ```
      </Tab>
      <Tab label="Examples">
        Here are some common usage patterns:

        - **Basic usage**: Simple component instantiation
        - **Advanced usage**: With custom configuration
        - **Integration**: Working with other libraries

        ```jsx
        // Basic usage
        <MyComponent />

        // With options
        <MyComponent options={{ debug: true }} />
        ```
      </Tab>
    </Tabs>

  </UiDisplay>

## With Icons

You can add icons to your tabs using the `icon` prop. Icons are sourced from Lucide and help make your tabs more visually distinctive.
